# HomeruGit

日付と時刻を自動でコミットメッセージとして使用し、GitHubにプッシュするシェルスクリプトです。

## 概要

`github-push.sh` は、Gitの操作を簡単にするための便利なスクリプトです。実行するだけで、現在の変更を自動的にステージング、コミット、プッシュまで一括で行います。

## 機能

- 📅 現在の日付と時刻を自動でコミットメッセージとして設定 (形式: `MM/DD HH:MM`)
- 📊 変更された行数を自動カウント
- 🎲 プッシュ成功時にランダムで励ましメッセージを表示
- ⚠️ エラーハンドリング機能付き

## 使い方

1. スクリプトに実行権限を付与:
```bash
chmod +x github-push.sh
```

2. スクリプトを実行:
```bash
./github-push.sh
```

## 出力例

```
💬 コミットメッセージ: 11/06 11:54
[main f7eec06] 11/06 11:54
 3 files changed, 93 insertions(+)

📤 'main'ブランチにプッシュ中...

------------------------------------------------
✅ GitHubへのプッシュが完了しました！

👏 今日も開発お疲れ様ー！
------------------------------------------------
```

## 注意事項

- このスクリプトは `main` ブランチにプッシュします
- 実行前に適切なブランチにいることを確認してください
- リモートリポジトリ `origin` が設定されている必要があります
