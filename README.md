# HomeruGit

GitHubを使ったの！？えらーい！

## 概要

`github-push.sh` は、Gitの操作を簡単にするための便利なスクリプトです。実行するだけで、現在の変更を自動的にステージング、コミット、プッシュまで一括で行います。

## 使い方

1. スクリプトに実行権限を付与:
```bash
chmod +x github-push.sh
```

2. スクリプトを実行:
```bash
./github-push.sh
```

3. 自動でpush:
```bash
✅ GitHubへのプッシュが完了しました！

👏 150行も進捗出せたね！今日も開発お疲れ様ー！
```

## 注意事項

- このスクリプトは `main` ブランチにプッシュします
- 実行前に適切なブランチにいることを確認してください
- リモートリポジトリ `origin` が設定されている必要があります

## ライセンス

自由にご利用ください。
